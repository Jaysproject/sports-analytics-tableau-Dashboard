# Sports Analytics Tableau Project

## Overview
This project explores player performance and match data using Tableau dashboards.

## Key Analysis
- Goalkeeper ranking based on save attempts
- Messi vs Ronaldo performance comparison
- Physical attributes vs performance metrics
- Interactive sports analytics dashboard

## Key Insights
- High save counts may reflect defensive pressure rather than goalkeeper quality
- Messi contributes more to build-up play, while Ronaldo focuses on finishing
- Physical attributes show weak correlation with performance metrics
